# **9. คำอธิบายรายวิชา (Course Description)**

รายวิชานี้มุ่งเน้นให้ผู้เรียนมีความเข้าใจในหลักการความมั่นคงไซเบอร์ ภัยคุกคาม การโจมตี และแนวทางการป้องกัน ครอบคลุมการวิเคราะห์ช่องโหว่ การประเมินความเสี่ยง การตรวจสอบทราฟฟิกเครือข่าย การใช้เครื่องมือด้านความมั่นคงไซเบอร์ การจัดการบันทึกเหตุการณ์ การใช้ระบบตรวจจับการบุกรุก ตลอดจนการประยุกต์ใช้ปัญญาประดิษฐ์เพื่อวิเคราะห์เหตุการณ์ผิดปกติเชิงลึก ผู้เรียนจะได้พัฒนาทักษะการคิดวิเคราะห์ การแก้ปัญหา และการปฏิบัติงานภายใต้หลักจริยธรรมด้านความมั่นคงไซเบอร์

# **10. ผลการเรียนรู้ของรายวิชา (Course Learning Outcomes: CLOs)**

### **ด้านความรู้ (Knowledge)**

**CLO1:** อธิบายหลักการพื้นฐานของความมั่นคงไซเบอร์ ภัยคุกคาม และการโจมตีได้อย่างถูกต้องและเป็นระบบ
**CLO2:** อธิบายการทำงานของโปรโตคอลเครือข่าย กลไกการควบคุมการเข้าถึง และมาตรฐานสากลด้านความมั่นคงไซเบอร์

### **ด้านทักษะ (Skills)**

**CLO3:** วิเคราะห์ทราฟฟิกเครือข่ายและตรวจจับพฤติกรรมผิดปกติด้วยเครื่องมือและวิธีการที่เหมาะสม
**CLO4:** ปฏิบัติการใช้เครื่องมือด้านความมั่นคงไซเบอร์ เช่น Wireshark, Suricata, Firewall, Log Pipeline ได้อย่างถูกต้อง
**CLO5:** ประยุกต์เทคนิคการเรียนรู้ของเครื่อง (Machine Learning) ในการตรวจจับมัลแวร์และความผิดปกติของระบบเครือข่าย

### **ด้านคุณธรรมและจริยธรรม (Ethics)**

**CLO6:** ปฏิบัติงานด้านความมั่นคงไซเบอร์ภายใต้จริยธรรม ความรับผิดชอบ และกฎหมายที่เกี่ยวข้อง

# **11. รายละเอียดการสอนประจำสัปดาห์ (12 Weeks Integrated Plan)**

### *(รองรับทั้งกลุ่มบรรยายเท่านั้น / กลุ่มบรรยาย + Workshop)*

| สัปดาห์ | หัวข้อการเรียนรู้                     | บรรยาย (Lecture)                                                    | ปฏิบัติการ (Workshop)                   | การประเมิน        |
| ------- | ------------------------------------- | ------------------------------------------------------------------- | --------------------------------------- | ----------------- |
| **1**   | Introduction to Cybersecurity         | หลักการ CIA/DAD, Threat–Vulnerability, Security as Asset Protection | วิเคราะห์เหตุการณ์โจมตีจริง             | Quiz + รายงาน     |
| **2**   | Network & Protocol Fundamentals       | IP, Subnet, OSI/TCP-IP, Protocol Weaknesses                         | วิเคราะห์ Packet Flow                   | Worksheet         |
| **3**   | Traffic Analysis with Wireshark       | Packet Dissection, Filters, PyShark                                 | Lab Wireshark + PyShark Script          | Lab Report        |
| **4**   | Threats, Attacks & Troubleshooting    | Malware, DoS, MITM, ARP Spoofing                                    | จำลอง ARP Spoofing + การตรวจจับ         | Quiz + Lab        |
| **5**   | Firewall Concepts                     | ACL, Stateful Inspection, NAT/PAT                                   | ตั้งค่า ACL & Firewall ใน Packet Tracer | Lab Assignment    |
| **6**   | LAN Security                          | VLAN, Port Security, DHCP Snooping, DAI                             | จำลอง LAN Attack Surface                | Lab               |
| **7**   | Malware Detection with ML             | Malware Taxonomy, PE Features, ML Algorithms                        | Train Model + Performance Analysis      | ML Lab            |
| **8**   | Traffic Anomaly Detection with ML     | NetFlow, Features, Isolation Forest                                 | Build Flow Dataset + Anomaly Detection  | Coding Assignment |
| **9**   | Log & Audit Log Management            | Log Standards, ELK Stack, Challenges                                | FastAPI Log + React Dashboard           | Log Lab           |
| **10**  | IDS/IPS with Snort & Suricata         | Rule Syntax, EVE JSON, Tuning                                       | Simulate Attacks + Analyze Alerts       | IDS Lab           |
| **11**  | Security Automation & Intelligent IDS | ML-enhanced IDS, WebSocket Pipeline                                 | Intelligent IDS Prototype               | Project Lab       |
| **12**  | Final Integration                     | SOC Concepts, Summary                                               | Practical Exam + Final Presentation     | Practical Exam    |
