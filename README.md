# **9. คำอธิบายรายวิชา (Course Description)**

รายวิชานี้มุ่งเน้นความรู้และทักษะด้านความมั่นคงไซเบอร์ ครอบคลุมหัวข้อพื้นฐานและเชิงลึก ได้แก่ ภัยคุกคามทางไซเบอร์ การโจมตีระบบคอมพิวเตอร์ การวิเคราะห์ความเสี่ยงและการจัดการความเสี่ยง กลไกและเทคโนโลยีการป้องกัน การบริหารนโยบายความมั่นคง มาตรฐานสากลด้านความปลอดภัย นิติคอมพิวเตอร์ กฎหมายและจริยธรรมทางไซเบอร์ รวมถึงการฝึกปฏิบัติด้านการตรวจสอบและวิเคราะห์เหตุการณ์ด้านความมั่นคงไซเบอร์

---

# **10. ผลการเรียนรู้ของรายวิชา (Course Learning Outcomes: CLOs)**

### **ด้านความรู้ (Knowledge)**

**CLO1:** อธิบายหลักการพื้นฐานของความมั่นคงไซเบอร์ ภัยคุกคาม และการโจมตีได้อย่างถูกต้อง
**CLO2:** อธิบายมาตรการควบคุมความมั่นคงไซเบอร์ มาตรฐานสากล และกรอบนโยบายความปลอดภัยได้อย่างเป็นระบบ

### **ด้านทักษะ (Skills)**

**CLO3:** วิเคราะห์ความเสี่ยงและระบุช่องโหว่ของระบบสารสนเทศได้อย่างเหมาะสม
**CLO4:** ปฏิบัติการใช้เครื่องมือด้านความมั่นคงไซเบอร์ในการตรวจสอบภัยคุกคามพื้นฐานได้

### **ด้านคุณธรรมและจริยธรรม (Ethics)**

**CLO5:** ประพฤติตนตามจริยธรรมในการทดสอบระบบ การใช้ข้อมูล และการปฏิบัติงานด้านความมั่นคงไซเบอร์

# **11. การเชื่อมโยง CLO – วิธีสอน – การประเมินผล**

| CLO  | วิธีการจัดการเรียนรู้ (TLA)  | การประเมินผล (Assessment)     |
| ---- | ---------------------------- | ----------------------------- |
| CLO1 | บรรยาย, กรณีศึกษา            | แบบทดสอบทฤษฎี                 |
| CLO2 | บรรยายเชิงวิเคราะห์, อภิปราย | แบบทดสอบและงานวิเคราะห์เอกสาร |
| CLO3 | วิเคราะห์สถานการณ์, Workshop | งานวิเคราะห์ช่องโหว่ / Lab    |
| CLO4 | Lab ปฏิบัติจริง              | การประเมินผลงานปฏิบัติ        |
| CLO5 | อภิปราย, สังเกตพฤติกรรม      | ประเมินพฤติกรรมและวินัย       |
